# KL Online Banking System

A modern online banking web application built using React, TypeScript, and Vite. The project provides a simple banking interface for users to manage transactions, view account information, and perform common banking operations through an intuitive dashboard.

## Features

* User Authentication (Login & Sign Up)
* Dashboard Overview
* Deposit and Withdraw Funds
* Money Transfer Functionality
* Transaction History
* Success Transaction Confirmation
* Responsive User Interface
* About Page
* Docker Support
* Vercel Deployment Ready

## Tech Stack

### Frontend

* React
* TypeScript
* Vite
* CSS

### Deployment

* Vercel
* Docker

## Installation

Clone the repository:

```bash
git clone https://github.com/eswar1795/KL-Online-Banking-System.git
cd KL-Online-Banking-System
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Deploy on Vercel

```bash
npm install -g vercel
vercel
```
